# 基于 Qt 的学生信息管理系统

## 一、项目简介

本项目是一个基于 Qt 开发的学生信息管理系统，主要用于完成学生信息的录入、查询、修改、删除、文件保存与读取等功能。

基础功能：
1. 记录课程成绩并进行GPA计算
2. 记录成就（奖项，班级代表，志愿者）
3. 导出一份完整的个人简历

创新功能：调用api让AI生成一个个人简历网页

## 五、 👥 小组分工（基于 MVC 架构）

本项目采用 MVC（Model-View-Controller）架构进行分层设计，并按照模块职责进行团队分工，具体如下：

| 成员 | 角色定位 | 负责模块 | 核心职责 |
|---|---|---|---|
| 郑润泽 | 系统架构师（System Architect） | 架构设计 | 负责整体系统架构设计；定义 MVC 分层结构；设计 Model、View、Controller 之间的接口规范；确保系统低耦合、高内聚 |
| 徐乐天 | 核心算法工程师（Model - Logic） | GPA计算模块 | 负责 GPA 计算模型设计与实现；处理学分与绩点换算逻辑；提供统一计算接口 |
