# PersonalDevPlan 使用说明（Mac 版）

## 安装

双击 `PersonalDevPlan-macOS.dmg`，把里面的 app 拖到 Applications 或桌面即可。

## 五个标签页

| 标签页 | 功能 |
|--------|------|
| **学业看板** | 课程成绩表 + GPA 计算 + 学期趋势图 |
| **个人信息** | 照片、教育经历、技能、工作经历 |
| **标准简历** | HTML 简历预览 |
| **AI 助手** | AI 对话助手（需要 DeepSeek API Key，没有则不可用） |
| **退出登录** | 退出当前账号 |

## 卸载

把 app 拖到废纸篓即可。数据目录在 `~/Library/Application Support/StudentDev/`，如需彻底清除可手动删除该目录。
